כלי עיצוב AI חדש

Anthropic משיקים לראשונה כלי עיצוב מבוסס AI שנבנה ישירות לתוך Claude.

בשפה פשוטה:

עם תיאור בשפה טבעית בלבד, אפשר ליצור אתרים, מצגות, דפי נחיתה ואבי טיפוס למוצרים. בלי צורך בידע עיצובי או כלים כמו Figma.

אתרים מלאים
בניית אתרים שלמים מתיאור טקסטואלי. HTML, CSS, JS מוכן להפצה
מצגות
יצירת מצגות מקצועיות מטקסט פשוט. מתחרה ב-Gamma ו-Google Stitch
דפי נחיתה
Landing pages עם עיצוב מודרני, responsive, מוכן לקמפיינים
אבי טיפוס
Prototypes למוצרים דיגיטליים. מרעיון לעיצוב עובד בדקות
מתחרים: הכלי מתחרה ישירות ב-Adobe, Figma, Gamma, Google Stitch ו-Lovable. זה המעבר הראשון של Anthropic מצ׳אט וכלי פיתוח לעולם העיצוב.

Deprecation: מה יוצא משימוש

חשוב לדעת

Claude Opus 4 המקורי (claude-opus-4-20250514) הוצא משימוש. הפסקת שירות מלאה ב-15 ביוני 2026. Anthropic ממליצים לעבור ל-Opus 4.7 (או לפחות 4.6).

שאלות נפוצות

שאלות ותשובות

מה ההבדל בין Opus 4.7 ל-Opus 4.6?
Opus 4.7 מוסיף כלי עיצוב AI חדש, Claude Code 2.0 עם Desktop App מחודש, פיצ׳ר Routines, ושיפורי ביצועים ב-Finance Agent ו-GDPVal. המחיר וחלון ההקשר נשארו זהים.
האם צריך לשלם יותר על Opus 4.7?
לא. המחיר זהה ל-Opus 4.6: $5 למיליון טוקני קלט ו-$25 למיליון טוקני פלט. אפשר גם להוריד עלויות עם Prompt Caching (90% הנחה) ו-Batch API (50% הנחה).
מה זה Claude Mythos?
Mythos הוא מודל חזק יותר מ-Opus 4.7 שנמצא בשלב בדיקה סגור. הוא לא זמין לציבור ונבדק רק על ידי 52+ ארגונים תחת Project Glasswing לצורך מחקר אבטחה.
מה לעשות עם Claude Opus 4 הישן?
לעבור ל-Opus 4.7 בהקדם. המודל המקורי (claude-opus-4-20250514) נסגר ב-15 ביוני 2026. Anthropic ממליצים לעדכן את ה-API calls שלכם עכשיו כדי למנוע הפתעות.

רוצים לנסות בעצמכם?

פתחו את Claude Code והתחילו לעבוד עם המודל החדש. זה קל, מהיר ושווה כל דקה.
